# Changelog

## 1.5.0.0
Key Info
 * NOT save-safe.
 * BSA'd the entire list, load time to the main menu should be lower.
 * Several gameplay mechanic changes, see the [gameplay guide](https://twistedskyrim.com/#gameplay) for more info.
 * All documentation has now been moved to the [website](https://twistedskyrim.com/), information found on the github page should be considered out of date.

<Details>
<summary>Patch Notes</summary>

### Bug Fixes
- Broken LOTD museum lod
- Floating objects outside of dragonsreach
- Picking flowers stuck issue
- Fixed head/neck discolorations and missing heads
- NFF Outfit crashing
- A couple landscape texture seams
- Fixed selling item crash

### Changes
- Perk system has been overhauled with Paragon, building ontop of Adamant -> check the gameplay guide for more information
- [Added an equipment degredation system](https://www.nexusmods.com/skyrimspecialedition/mods/109899)
- Static Skill releveling was added in place of the current leveling system
- Added kill exp with experience
- A lot of changes to the ENB. Should overall look better than previously.

### Known Issues
- There may be some clipping or other oddities around the museum in solitude (LOTD). I made it as good as it can be while I wait for the official patch to be released.
